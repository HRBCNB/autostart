# 自动启动工具 - 你能干什么？

## 工具功能介绍

这个自动化工具专为原神玩家设计，可以自动化执行 BetterGI 的各种任务。以下是工具的主要功能：

### 🖥️ 桌面管理
- **显示桌面**：自动最小化所有窗口，确保桌面清洁
- **窗口控制**：为自动化操作创建理想的桌面环境

### 🔊 音频控制
- **智能静音**：自动静音默认扬声器，避免游戏声音干扰
- **音量管理**：支持静音和取消静音功能

### ⚙️ 进程管理
- **智能清理**：自动关闭以下进程，避免冲突：
  - 微星小飞机 (MSI Afterburner)
  - 原神游戏进程 (Genshin Impact)
  - BetterGI 进程
  - Snipaste 截图工具
- **优雅终止**：安全地结束进程，避免数据丢失

### 🎮 游戏自动化
- **自动启动 BetterGI**：根据配置文件自动启动 BetterGI
- **鼠标自动化**：执行预设的鼠标点击序列
- **任务配置**：支持不同的任务配置：
  - 📅 **每日任务**：自动执行日常任务
  - 🌾 **自动锄地**：自动收集资源

### 🛠️ 配置工具
- **鼠标位置记录器**：帮助记录准确的点击坐标
- **灵活配置**：通过 `config.txt` 轻松自定义任务流程
- **多配置支持**：支持多种预设配置，适应不同需求

### ⏰ 定时执行
- **Windows 任务计划程序集成**：支持定时自动执行
- **无人值守运行**：可在指定时间自动运行，无需人工干预

### 🔧 技术特性
- **权限管理**：支持管理员权限运行，确保功能完整性
- **错误处理**：内置错误处理机制，提高稳定性
- **中文友好**：完全支持中文界面和配置

# 使用方法

## 一、配置 `config.txt` 文件

`config.txt` 文件格式如下：

- **第一行**：`BetterGI.exe` 的完整安装路径  
- **后续每一行**：一次鼠标点击位置，格式为 `x,y`，分别为横纵坐标

### 步骤说明：

1. **找到 BetterGI 的安装目录**  
   右键快捷方式 → 打开文件所在位置  
   ![](https://gitee.com/hrbcnb/typora_pic/raw/master/img/20250529110111272.png)

2. **复制绝对路径**  
   ![](https://gitee.com/hrbcnb/typora_pic/raw/master/img/20250529110130326.png)

3. **打开 `config.txt` 文件**  
   将复制的路径粘贴在第一行，并添加 `\BetterGI.exe`，例如：

   ```
   D:\Program Files\BetterGI\BetterGI.exe
   ```

4. **以管理员身份运行 `mouse_position_recorder`**  
   右键 → 以管理员身份运行  
   > ⚠ 如果不是管理员身份运行，当焦点不在终端时将无法获取坐标

   ![](https://gitee.com/hrbcnb/typora_pic/raw/master/img/20250529110150733.png)

5. **启动 BetterGI**  
   若弹出“是否允许该程序更改你的设备”，直接点击“是”，**该点击不需要记录坐标**

6. **逐一记录点击坐标**  
   打开 BetterGI 后，依次记录每一个有效点击位置  
   
   > （获取的坐标点有可能会有没有用的，注意观察有效点击对应的坐标）
   
7. **最终配置示例**  
   ![](https://gitee.com/hrbcnb/typora_pic/raw/master/img/20250529110219701.png)

- 支持每日任务和资源收集自动化
- 定时执行，无需人工干预

🛡️ **安全保障**：
- 优雅的进程管理，避免数据丢失
- 智能错误处理，确保系统稳定
- 支持管理员权限，功能完整可靠

⚡ **效率提升**：
- 一键配置，终身受益
